Implementation Approach

In this project, we are following Incremental Development Model. In first Iteration, firstly we had visualized the dataset, derived the correlation between the features using Orange Frame work and checked the dependency of the features on the target variable. After First Incrementation is over, In second iteration, we had build the algorithm and trained the model using smaller dataset i.e. 100,000 records using Keras on the top of Tensor Flow. In third Iteration, we have trained the model by feeding multi-million(40mn) records dataset, for getting a good fit and higher accuracy. In fourth iteration, we have developed the front end for flagging the illegitimate user, using HTML,PHP, CSS and other Web Based technologies.
